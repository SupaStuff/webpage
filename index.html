EBShare

An Ebook sharing system.

This website is a class project that me, Moustafa Elshaabiny, and Abhishek Samdaria worked on during the Fall of 2015. We designed an online web platform that provides a catalog of available electronic books. Each E-book has a cover-page, a .txt file including the contents of the book and a summarized description to be displayed in the catalog and the number of points needed to read the book for a certain time.

myFutureHS

A High School Search App

myFutureHS is a search application that finds High Schools based on criteria such as demographics, excellence, safety, and ELL availability. The results come from database queries on data provided by NYC DOE. Most of the features are implemented. The results are displayed as pins on the map and an expandable list under the categories. Results are fetched in real time, so any time an option is changed, the results are automatically refreshed.
